1.ALLEN과 부서가 같은 사원들의 사원명, 입사일을 출력하되 높은 
  급여순으로 출력하세요.
--------------------------------------------------------------------------------------
SELECT  ENAME, HIREDATE
FROM EMP
WHERE DEPNTO=(
SELECT DEPTNO FROM EMP WHERE ENAME='ALLEN')
ORDER BY SAL DESC; 

--------------------------------------------------------------------------------------
2.최대급여를 받는 사원보다 입사일이 늦은 사원의 
  이름, 입사일을 출력하세요.
--------------------------------------------------------------------------------------
SELECT ENAME, HIREDATE 
FROM EMP
WHERE HIREDATE > (
 SELECT HIREDATE FROM EMP 
 WHERE SAL=(SELECT MAX(SAL) FROM EMP)
);


--------------------------------------------------------------------------------------  
3.이름에 "T"자가 들어가는 사원들의 급여의 합을 구하세요.
--------------------------------------------------------------------------------------
SELECT SUM(SAL) FROM EMP 
ENAME  LIKE '%T%';

--------------------------------------------------------------------------------------
4.20번 부서의 최고 급여보다 급여가 높은 사원의 
   사원번호,사원명,급여를 출력하세요.
--------------------------------------------------------------------------------------
SELECT EMPNO, ENAME, SAL
FROM EMP 
WHERE SAL >(SELECT MAX(SAL) FROM EMP 
                       WHERE DEPNTO=20);

----------------------------------------------------------------------
5.SCOTT의 급여에서 1000 을 뺀 급여보다 적게 받는 사원의 
  이름,급여를 출력하세요.
----------------------------------------------------------------------
SELECT ENAME, SAL 
FROM EMP 
WHERE SAL < (SELECT SAL -1000 FROM EMP 
                         WHERE ENAME='SCOTT');
